Binary and Unary

Binary - operator requires two operands

Unary - operator requires one operands on either side

Operators

Assignment - assigns a value to its left operand based on the value of its right operand.

Comparison Operators

compares its operands and returns a logical value based on whether the comparison is true

Loops

offer a quick and easy way to do something repeatedly. Various loop mechanisms offer different ways to determine the start and end points of the loop.

for Statement

A for loop repeats until a specified condition evaluates to false

Initial Expression - expression usually initializes one or more loop counters, and can declare variables

Condition Expression - expression is evaluated. If the value of conditionExpression is true, the loop statements execute. Otherwise, the for loop terminates.
