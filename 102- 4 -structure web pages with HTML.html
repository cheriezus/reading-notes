Structure Web Pages with HTML

Wrieframe

Wireframing is a practice used to define and plan the design and placement of contect for a website.

Always suggested that you draw out your website, and find where you would like to place informtaion on the website. Multiple boxes are used in order to outline the: Header, Footer, Images and Text.

HTML Basics

HTML is a markup language that is used to add structure to your website. HTML consists of "elements", that define what kind of content will be placed where. Elements consist of an opening and closing tag, with a character that tells you what kind of element it is ex: p = paragraph, = adding emphasis, that shows up with a bold font, () = adding an image.
